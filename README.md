# SoraWebui

## Quick Started

1. Clone project

```bash
git clone git@github.com:SoraWebui/SoraWebui.git
```

2. Install dependencies

```bash
cd SoraWebui && yarn
#or
cd SoraWebui && npm install
#or
cd SoraWebui && pnpm install
```

3. Run it

```bash
yarn dev
#or
npm run dev
#or
pnpm dev
```

4. Open [http://localhost](http://localhost) with your browser to see it.
